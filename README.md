## Overview
**ofxSurfingPlayer** is a simple add-on for **openFrameworks** with a basic **BPM based** timer to trig **Bangs**.

## Features
- **ofParameter** based. 
- **ofxGui** or **ImGui** modes.  
- **Tap Tempo** Engine  
- Tick **Sounds**
- Selector to redirect the Bang to many targets.
